# spring-mysql-redis-cache
Spring Boot Mysql Redis  REST API Cache example

##
### Prerequisites
- JDK 1.8
- Maven
- Mysql
- Redis

## Quick Start

### Clone source
```
git clone https://github.com/jeonguk/spring-mysql-redis-cache.git
cd spring-mysql-redis-cache
```

###
```
mvn clean install
```
